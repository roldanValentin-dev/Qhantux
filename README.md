Qhantux: Soluciones Tecnológicas con Identidad
Bienvenido al repositorio del proyecto final para el curso de desarrollo web. Este proyecto es un sitio web estático para Qhantux, una empresa ficticia de soluciones tecnológicas que se destaca por combinar la innovación con un enfoque humano y local.

Acerca del sitio
El sitio web de Qhantux está diseñado para ser la carta de presentación de la empresa, mostrando sus valores, servicios y el equipo de profesionales detrás de las soluciones. El objetivo principal es comunicar la misión y visión de la compañía de forma clara y atractiva, utilizando un diseño moderno y responsive.

El proyecto cuenta con las siguientes secciones:

Inicio: Una introducción a la filosofía de la empresa.

Nosotros: Detalla la misión, visión, valores y presenta al equipo de trabajo.

Servicios: Describe las soluciones tecnológicas que ofrece Qhantux.

Tecnologías y Herramientas Utilizadas
Este proyecto fue desarrollado aplicando las siguientes herramientas y lenguajes para asegurar una estructura sólida, un diseño atractivo y un rendimiento óptimo:

HTML: Utilizado para la estructura semántica y la jerarquía de contenidos.

CSS: Implementación de propiedades como Flexbox y Grid para un maquetado flexible y adaptativo.

Sass: Preprocesador de CSS que me permitió organizar los estilos de manera modular y eficiente.

Bootstrap: Framework de CSS para acelerar el desarrollo del diseño responsive y asegurar la compatibilidad con diferentes dispositivos.

Optimización de Imágenes: Compresión y uso de formatos modernos para mejorar el tiempo de carga del sitio.

Git y GitHub: Control de versiones del proyecto.

Lo que Aprendí
Durante el desarrollo de este proyecto, puse en práctica y consolidé los siguientes conocimientos:

HTML y CSS: Aprendí a estructurar y dar estilo a un sitio web desde cero, entendiendo la base de toda la web.

Maquetación Responsive: Asegurar que el sitio se vea y funcione perfectamente en dispositivos móviles, tablets y computadoras de escritorio, utilizando una combinación de media queries, Flexbox y Grid.

SEO Básico: Optimizar la estructura del código, la jerarquía de encabezados (h1, h2, etc.) y el uso de atributos alt en las imágenes para mejorar la visibilidad en los motores de búsqueda.

Optimización del Rendimiento: Comprimir imágenes y organizar el código para garantizar una carga rápida y una experiencia de usuario fluida.

Flujo de Trabajo con Sass: Aprender a usar un preprocesador para escribir CSS de forma más limpia, escalable y organizada.
